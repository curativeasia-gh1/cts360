## Stack

- Pure HTML/CSS/JS — no build tools, no dependencies
- Shared nav & footer injected via `assets/js/components.js`
- Responsive: 1024px / 860px / 600px breakpoints
- Google Fonts: Inter + Syne
- Schema.org JSON-LD on homepage
